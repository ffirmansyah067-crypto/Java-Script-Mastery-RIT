Apa itu JavaScript?

JavaScript adalah bahasa pemrograman tingkat tinggi yang awalnya dikembangkan untuk membuat halaman web menjadi lebih interaktif. Diciptakan oleh Brendan Eich pada tahun 1995, JavaScript kini telah menjadi salah satu bahasa paling populer di dunia.

 Variabel, Tipe Data, dan Operator

Variabel

Variabel adalah tempat untuk menyimpan data. Dalam JavaScript modern, kita menggunakan tiga cara untuk deklarasi:

var: cara lama, memiliki scope function
let: deklarasi variabel yang bisa diubah nilainya
const: deklarasi variabel yang nilainya tetap (konstan)
=============================================================>

Tipe Data Primitif

String: Teks, diapit oleh tanda kutip
Number: Angka bulat atau desimal
Boolean: `true` atau `false`
Null: Nilai kosong secara sengaja
Undefined: Nilai belum ditentukan

Operator

Aritmatika: `+`, `-`, `*`, `/`, `%`
Perbandingan: `==`, `===`, `!=`, `!==`, `>`, `<`, `>=`, `<=`
Logika: `&&` (dan), `||` (atau), `!` (not)

=============================================================>

Struktur Kontrol (Percabangan & Perulangan)

Percabangan (Conditional Statement)

`if`, `else if`, `else`

`switch` Statement

Perulangan (Looping)
for, while, do while

=============================================================>
Fungsi: Deklarasi, Parameter, Return, Arrow Function (ES6)

Fungsi adalah blok kode yang dirancang untuk melakukan tugas tertentu dan bisa digunakan kembali.
Parameter: variabel input ke fungsi
Return: nilai yang dikembalikan oleh fungsi
