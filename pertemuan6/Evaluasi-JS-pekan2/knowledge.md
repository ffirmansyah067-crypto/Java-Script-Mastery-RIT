Apa itu Java Script
Java Script adalah bahasa pemrograman tingkat tinggi yang awalnya dikembangkan untuk membuat halaman web menjadi lebih interaktif. Diciptakan oleh Brendan Eich pada tahun 1995, JavaScript kini telah menjadi salah satu bahasa paling populer di dunia.


Lalu di Java Script ada bagian bagiannya seperti,

<strong>Variabel<br></strong>
Variabel adalah tempat untuk menyimpan data (wadah).

Lalu Ada tiga cara untuk mendeklarasikan Variabel, yaitu :<br>
Var: Cara lama, memiliki scope function.<br>
Let: Deklarasi variabel yang bisa diubah nilainya.<br>
Cons: Deklarasi variabel yang nilainya tetap (konstan).<br>

<strong>Tipe Data Primitif<br></strong>
String: Teks, diapit oleh tanda kutip
Number: Angka bulat atau desimal<br>
Boolean: `true` atau `false`<br>
Null: Nilai kosong secara sengaja<br>
Undefined: Nilai belum ditentukan
<br>

<strong>Operator<br></strong>
Aritmatika: `+`, `-`, `*`, `/`, `%`<br>
Perbandingan: `==`, `===`, `!=`, `!==`, `>`, `<`, `>=`, `<=`<br>
Logika: `&&` (dan), `||` (atau), `!` (not)

<strong>Struktur Kontrol<br></strong>

Ada dua macam yaitu:<br>

<strong>Percabangan (Conditional Statement)<br></strong>
`if`, `else if`, `else` , switch `Statement`<br>

<strong>Pengulangan (Looping)<br></strong>
for, while, do while<br>

<strong>Fungsi: Deklarasi, Parameter, Return, Arrow Function (ES6)<br></strong>

Fungsi adalah blok kode yang dirancang untuk melakukan tugas tertentu dan bisa digunakan kembali.<br>
Parameter: variabel input ke fungsi. <br>
Return: nilai yang dikembalikan oleh fungsi.<br>

<strong>Scope, Hoisting, dan Closure<br></strong>

Scope adalah area di mana variabel dan fungsi dapat diakses. <br>
Ada 2 jenis scope: global dan lokal.<br>
Scope global adalah variabel yang dapat diakses dari mana saja dalam kode.<br>
Scope lokal adalah variabel yang dideklarasikan di dalam fungsi.<br>

<strong>Hoisting<br></strong>

Hoisting adalah mekanisme JavaScript untuk mengangkat deklarasi variabel dan fungsi ke bagian atas scope-nya sebelum kode dieksekusi.<br>

<strong>Clousure<br></strong>
Closure adalah fitur di mana fungsi memiliki akses ke variabel dari scope induknya, bahkan setelah scope induknya selesai dieksekusi.

<strong>Operator Logika</strong><br>
&&  : Hasilnya true apabila kedua kondisinya sama sama true<br>
||  : Hasilnya true jika salah satu kondisinya true<br>
|   : Membalikkam nilai boolean<br>
!   : Bisa membalikkan fakta<br>
